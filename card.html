Card Cardiac

Card Cardiac is a fun and engaging Unity-based card-flipping game that tests players' memory and speed. Players aim to match pairs of cards with the fewest flips and the shortest time. The game features a user authentication system using Firebase for secure login and registration, along with a real-time leaderboard showcasing the top players.

Software Used

    Unity - Development of game interactions, UI/UX elements, and sprite integration.
    C# - Scripted logic, behaviors, and UI flow.
    Firebase - Backend support for authentication, real-time database, cloud functions, and data storage.
